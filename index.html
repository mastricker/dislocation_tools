Dislocation tools¶

This reposistory contains functions for inserting dislocations into crystals by imposing the anisotropic-elastic solution of the displacement field.

Documentation: https://wgnoehring.github.io/dislocation_tools/

Source: https://github.com/wgnoehring/dislocation_tools

Dependencies¶

The package requires :

  • numpy - http://www.numpy.org/

  • scipy - http://www.scipy.org/

  • sympy - https://www.sympy.org

Optional packages :

  • lammps - https://lammps.sandia.gov/doc/Python_run.html
